# CLink Menu 🍽️
### حاسبة تكلفة المطاعم الذكية

**Restaurant Food Cost Calculator — Powered by Daftra API**

---

## ما هو CLink Menu؟

تطبيق متكامل لحساب تكلفة الأطباق وإدارة هوامش الربح في المطاعم والكافيهات، مع تكامل مباشر مع منصة **دفترة ERP**.

## المميزات الرئيسية

- ✅ حساب Food Cost % لكل طبق فورياً
- ✅ ربط مباشر بمنتجات دفترة
- ✅ تحديث أسعار المكوّنات من فواتير المشتريات
- ✅ دعم 7 عملات عربية (مصر، السعودية، الإمارات...)
- ✅ تقارير شاملة وتنبيهات ذكية
- ✅ واجهة عربية RTL كاملة

## التقنيات المستخدمة

- React 18 + Vite
- Daftra REST API
- CSS-in-JS
